# 🎓 Educa – LMS Frontend Template

A modern and responsive Learning Management System (LMS) frontend template designed for educational platforms. This project provides a clean UI with course listings, tutor profiles, dashboard sections, and interactive components.

---

## 🚀 Features

- Responsive and modern UI design
- Dashboard with quick statistics (likes, comments, playlists)
- Course listing section with tutor information
- Sidebar navigation menu
- Search functionality UI
- User profile dropdown
- Dark/Light mode toggle (UI ready)
- Category and topic sections
- Clean and structured layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
